## О проекте

В данном репозитории реализован **полностью работающий CI/CD пайплайн** для C++ HTTP-приложения с использованием **GitHub Actions**.  
Пайплайн включает:

- Автоматическую сборку и тестирование (unit, integration, load)
- Статический анализ кода (clang-tidy)
- Сборку Docker-образа и публикацию в GitHub Container Registry
- Автоматический деплой в **staging** (при push в `main`)
- Ручной деплой в **production** по стратегии **blue-green** (с выбором версии)
- **Откат** до любой предыдущей версии через отдельный workflow
- Мониторинг (метрики `/metrics`), логирование и уведомления в виде артефактов
